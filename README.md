##### select快速选择
本插件为jquery插件
需要依赖template



##### install
~~~
<select name="" id="testSelect" data-quick-select>
        <option value="104">xxx</option>
        <option value="105">yyy</option>
        <option value="106">zzz</option>
</select>
//调用
$("select[data-quick-select]").QuickSelect();
~~~
